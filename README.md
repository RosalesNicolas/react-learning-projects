# React Learning Projects

Collection of React projects and exercises for practice and portfolio (course).

## What’s inside

This repository contains multiple small projects. Each folder is a self-contained exercise/app with its own goal.

- **01-reforzamiento** — fundamentals practice (JS/TS + React basics)
- **02-first-steps** — first React components, props, state, and testing basics
- **03-gifs-app** — GIF search app (course project)

> Note: names and content may evolve as I progress through the course.

## Tech stack

- React

- Vite
- Testing: Vitest + React Testing Library (where applicable)
